# 自定义音效（可选）

软件默认使用代码合成的音效 + 系统朗读，无需任何音频文件即可工作。
如果您希望换成更逼真的真实掌声/欢呼，可以把对应文件放到此目录下，刷新页面即可生效：

| 文件名         | 用在哪里                          | 没有则用什么代替              |
|----------------|-----------------------------------|-------------------------------|
| `cheer.mp3`    | 连对 5/7 题、整组高分时的欢呼     | 代码合成的"成功号角"          |
| `applause.mp3` | 整组结束（≥80%）的掌声            | 代码合成的白噪声脉冲          |
| `ding.mp3`     | 单题答对的"叮"声                  | 代码合成的"啵"声              |

## 推荐来源

任何 CC0 / 公共领域 / 您自有版权的音频都可以：

- [Freesound.org](https://freesound.org/) — 注册后可下载，注意筛选 CC0 协议
- [Pixabay Sound Effects](https://pixabay.com/sound-effects/) — 免费可商用
- 自己录一段也可以（孩子听到妈妈的鼓励声会更开心）

## 建议规格

- 格式：MP3（兼容性最好）
- 长度：1~3 秒
- 大小：< 50KB / 个
- 音量：录制或导出时不要削顶，软件里会再调到 50~60%

## 如何禁用某个音

不想用某种音效，删掉对应文件即可，软件自动回退到合成版本。
也可以点右上角的 🔊 按钮整体静音。
